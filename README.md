# 🎵 Musique Magasin

Petite application **100 % locale** pour un magasin :

- 🔊 **Musique non-stop** (lue depuis le dossier `music` du PC)
- 🎤 **Micro « parler »** par-dessus la musique (effet talkie-walkie)
- 🕐 **Annonce de fermeture automatique** chaque jour, à voix haute

Le **PC** (Windows) fait tourner le serveur ; l'**iPad** à la caisse ouvre
l'interface par le Wi-Fi. Rien n'est hébergé en ligne.

## Démarrage rapide
1. Installer [Node.js](https://nodejs.org) (version LTS) sur le PC.
2. Mettre ses `.mp3` dans le dossier `music`.
3. Double-cliquer sur **`DEMARRER-WINDOWS.bat`**.
4. Sur l'iPad : installer le certificat puis ouvrir l'adresse `https://…`
   affichée.

👉 Détails complets dans **[GUIDE.md](GUIDE.md)**.
